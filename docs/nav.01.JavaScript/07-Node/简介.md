# 简介

- Node.js 是一个开源与跨平台的 JavaScript 运行时环境
- 在浏览器外运行 V8 JavaScript 引擎（Google Chrome 的内核）
- 特点
  - 非阻塞 io
    - 在做I/O操作的时候不会造成任何的阻塞，当完成之后，以事件的形式通知执行操作
  - 事件驱动
    - 事件循环

## 优缺点

优点：

- 适合处理高并发场景
- 适合I/O密集型应用，应用在运行极限时，CPU占用率仍然比较低，大部分时间是在做 I/O 硬盘内存读写操作

因为Nodejs是单线程，带来的缺点有：

- 不适合CPU密集型应用
- 只支持单核CPU，不能充分利用CPU
- 可靠性低，一旦代码某个环节崩溃，整个系统都崩溃
